Data Sources and Notes

DocID: This is an autogenerated value that is created when a new record is added.
EventID: This corresponds to the primary key(s) of the Autosalt Summary table and tells us which event the documents relate to.
SiteID: This corresponds to the primary key(s) of the Autosalt Summary table and tells us which event the documents relate to.
Link: A google drive link that brings you to the discharge calculations excel sheet.
  • There have been several different versions of the excel sheet over the project's history and some events have more than one excel sheet depending on the number of sensors at the site.
Checked: Indicates if the document been checked by a user.
Edits Made: Indicates if there were any changes made to the discharge calculations when the document was checked.
  • Prior to May 2021, all documents are marked Y which may be untrue for some documents. The lack of records prevents us from having more precise information prior to 2021.

Event Date: Date when the event took place (YYYY-MM-DD).

Data Specifications

Checked --> Allowed values: Y(Yes), N(No)
Edits Made --> Allowed values: Y(Yes), N(No)

Data Constraints

1. If a document has been checked (i.e. Checked =Y) then Edits Made can not be null
2. When entering the Link into the database it must be entered in HTML format in order to display the link properly
