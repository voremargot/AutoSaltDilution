Data Sources and Notes

All data comes from the csv that is used to create the rating curves.

RCManualID: This is an autogenerated value that is created when a new record is added.
Site ID: This corresponds to the primary key in the site description table and represents which site the discharge collection happened at.
MDisID: This corresponds to the primary key in the manual discharge table and defines which manual discharge collection event we are referencing.
RCID: This corresponds to the primary key in the RC Summary table and defines which version of the rating curve the value is included in.
EventNo: This identifies which precipitation event the discharge value is part of. This value is selected by the user when the rating curve is created and is only avaliable for RC version 4 and above.
  • The majority of manual discharge events did not occur during a precipitation event, represented by a zero.
