Data Sources and Notes

CalResultsID (PK): This is tan autogenerated value.
CalEventID (FK): This cooresponds to the primary key in the calibration events table and specifies which calibration event the calibration results are from
SiteID (FK): This cooresponds to the primary key in the site description table and specifies which site the calibration results are for
SensorID (FK): This cooresponds to the primary key in the sensors table and specifies which sensor was being calibrated
Trial Number: The trial that the calibration factor is from
Temp: Temperature of stream during the calibration event. This is the average of the stream temperatures for a given sensor on the completed field form.
CF Value: The calibration factor (CF) as calculated in the field. The CF value from the completed field form is scaled by 10^6 before being entered into the database.
Per Err: The percent error of the calibration factor. This value is obtained from the completed field form.
Flags: Designated flags mark common errors seen in the CF values
Notes: Notes about the specific EC measurement
Link: Google Drive link to the original,completed field form where the infomation in the table was gathered from

Data Specifications

Temp --> degrees C
CF Value --> Scaled by 10^6
Per Err --> %

Flags

Flags
L --> Unusally low CF values (CF < 2.20)
H --> Unusally high CF values (CF >2.9)

Data Constraints

1. When entering the Link into the database it must be entered in HTML format in order to display the link properly
