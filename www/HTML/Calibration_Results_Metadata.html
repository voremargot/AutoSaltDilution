Data Sources and Notes

CalResultsID: This is an autogenerated value that is created when a new record is added.
CalEventID: This corresponds to the primary key in the calibration events table and specifies which calibration event the calibration results are from.
SiteID: This corresponds to the primary key in the site description table and specifies which site the calibration results are for.
SensorID: This corresponds to the primary key in the sensors table and specifies which sensor was being calibrated.
Trial Number: The trial that the calibration factor is from.
Temp: The stream temperature during the calibration event. This is the average of the stream temperatures for a given sensor on the completed field form.
CF Value: The calibration factor (CF) as calculated in the field. The CF value (extracted from the completed field form) is scaled by 10^6 before being entered into the database.
Per Err: The percent error of the calibration factor. This value is extracted from the completed field form.
Flags: Designated flags that mark common errors seen in the CF values.
Notes: Notes about the specific EC measurement.
Link: A google drive link to the original, completed field form where the information in the table was gathered from.

Data Specifications

Temp --> degrees C
CF Value --> Scaled by 10^6
Per Err --> %

Flags

Flags
L --> Unusually low CF values (CF < 2.20)
H --> Unusually high CF values (CF >2.9)

Data Constraints

1. The Link must be entered into the database in HTML format in order for the hyperlink to display properly.
