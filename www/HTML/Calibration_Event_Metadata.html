Data Sources and Notes

CalEventID (PK): This is an autogenerated value.
PeriodID (FK): This cooresponds to the primary key in the barrel period table and specifies which barrel period the calibration event took place in
SiteID (FK): This cooresponds to the primary key in the site description table and specifies which site the calibration event occured at
Date: When did the calibration event occur? This data is taken from the completed field sheet
PMP: Specifies when in the barrel period the calibration event occurred. This data is taken from the completed field sheet
Trail: The number of calibration trials that occurred during the calibration event
Location: Specifies if the calibration event occurred on the autodump site or in the lab. This data is taken from the completed field sheet

Data Spcifications

PMP --> Accepted Values: Pre, Post, Mid, Unknown
Location --> Accepted Values: On Site, Lab

Data Constraints

1. PMP and Location can only be recorded as one of the accepted values. If different values need to be added, the data type will need to be changed within the database scheme
