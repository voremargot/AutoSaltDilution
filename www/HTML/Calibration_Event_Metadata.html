Data Sources and Notes

All data in the table come from the CF excel sheets that are filled out in the field.

CalEventID: This is an autogenerated value that is created when a new record is added.
PeriodID: This corresponds to the primary key in the barrel period table and specifies which barrel period the calibration event took place in.
SiteID : This corresponds to the primary key in the site description table and specifies which site the calibration event occurred at.
Date: The date when the calibration event occur (YYYY-MM-DD).
PMP: This specifies when in the barrel period the calibration event occurred (Pre fill, Mid period, Post fill).
Trail: The number of calibration trials that occurred during the calibration event.
Location: Specifies if the calibration event occurred at the field site or in the lab.
Temp: The average water temperature during the CF measurements.

Data Specifications

PMP --> Accepted Values: Pre, Post, Mid, Unknown
Location --> Accepted Values: On Site, Lab
Temp --> C

Data Constraints

1. PMP and location can only be recorded as one of the accepted values. If different values need to be added, the data type will need to be changed within the database scheme
