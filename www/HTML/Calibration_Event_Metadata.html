Data Sources and Notes

CalEventID: This is an autogenerated value that is created when a new record is added.
PeriodID: This cooresponds to the primary key in the barrel period table and specifies which barrel period the calibration event took place in.
SiteID : This cooresponds to the primary key in the site description table and specifies which site the calibration event occured at.
Date: This specifies when the calibration event occur. The date is extracted from the completed field sheet.
PMP: This specifies when in the barrel period the calibration event occurred. This data is extracted from the completed field sheet.
Trail: This is the number of calibration trials that occurred during the calibration event.
Location: This specifies if the calibration event occurred at the autodump site or in the lab. This information is extracted from the completed field sheet

Data Spcifications

PMP --> Accepted Values: Pre, Post, Mid, Unknown
Location --> Accepted Values: On Site, Lab

Data Constraints

1. PMP and location can only be recorded as one of the accepted values. If different values need to be added, the data type will need to be changed within the database scheme
