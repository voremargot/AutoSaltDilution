Data Sources and Notes

Note that this table has 2 primary key columns

EventID: The EventID value is generated by the autosalt system when a dump event occurs. This ID value comes from the raw dump event file and is a primary key column.
SiteID: This corresponds to the primary key in the site description table, specifying which site the dump event took place at AND is a primary key column.
PeriodID: This corresponds to the primary key in the barrel periods table and specifies which barrel period the dump event took place during.
Date: This is the date the dump event occurred on. The date comes from the summary file of dump events in the Hakai repository.
Temp: This is the stream temperature at the start of the dump event. The information comes from the summary file of dump events in the Hakai repository.
Start Time: The time when the dump event began. The time comes from the summary file of dump events in the Hakai repository.
Stage DR: This is the stream stage at the time of solution release. The data comes from the summary file of the dump events in the Hakai repository
Stage Min: The minimum stage of the stream over the course of the dump event. The minimum value represents the lowest stage value between the start and end of the salt curve.
Stage Max: The maximum stage of the stream over the course of the dump event. The maximum value represents the highest stage value between the start and end of the salt curve.
Stage Std: This is the standard deviation of the stage values between the start and end of the salt curve.
Stage Dir: This specifies how the stage is changing over the span of the dump event.
Salt Volume: This is the amount of salt solution that is dumped into the stream during the event. This data comes from the summary file of dump events in the Hakai repository.
Discharge Avg: The average of error-free, individual discharge calculations (see all discharges table).
Uncert: The uncertainty of the discharge average.
Flags: Highlights common errors seen within the dump event data.
ECb: Highlights changes in the base EC (ECb) over the duration of the salt curve.
Notes: Specific comments about the overall discharge average or dump event.

Data Specifications

Temp --> degrees C
Stage DR --> cm
Stage Average --> cm
Stage Min --> cm
Stage Max --> cm
Stage Std --> cm
Salt Volume --> L
Discharge Avg --> m3/s
Uncert --> %

Flags

Flags
ND -> No raw data was found for the dump event
NW -> Raw data was avaliable but there no salt wave occurred
PW -> Only a partial wave was recorded thus no discharge was calculated
ECb
R-> Base EC is rising over the span of the dump event
F-> Base EC is falling the span of the dump event
C-> Base EC is constant over the span of the dump event
Notes Offset -> Indicating that 2 or more of the salt wave peaks are offset from eachother by more than 35 seconds
