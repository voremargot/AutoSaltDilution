Data Sources and Notes

RCAutoID (PK): This is an autogenerated value.
Site ID (FK): This cooresponds to the primary key in the autosalt summary table and represents which site the dump event happened in.
EventID (FK): This cooresponds to the primary key in the autosalt summary and defines which dump event we are referencing.
RCID: This cooresponds to the primary key in the RC Summary table and defines which version of the rating curve the value is included in.
EventNo: This identifies which precipitation event the discharge value is part of. This value is selected by the user when the rating curve is created and is only avaliable for RC version 4 and above.
