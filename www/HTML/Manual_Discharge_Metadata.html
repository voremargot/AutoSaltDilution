Data Sources and Notes

MDisID (PK): This is an autogenerated value.
SiteID (FK): Thsi cooresponds to the primary key of the site description table and defines which site the manual discharge measurement occurred at.
Date: The date when the the discharge measurement was taken.
Time: The time when the discharge measurement was taken.
Instream Loc: The instream location where the discharge measurement was taken.
Stage: The stream stage at the time of discharge measurement. This is taken from the raw stage data.
Discharge The discharge of the stream.
Uncertainty The percent error in the discharge measurement.
Method The manual method used for taking the discharge measurement.
Comment: General comments about the collection of the dischrage measurement.
Images: An image of the site where the discharge value was taken.
Weblink: A google drive link that directs you to the excel sheet used in the field to collect the discharge measurment.

Data Specifications

Stage --> cm
Discharge --> m3/s
Uncertainty --> %
Method --> Accepted values: Propeller, Flow Tracker, Salt

Data Constraints

1. Discharge needs to be positive.
2. Stage needs to be positive.
3. Uncertainty needs to be positive.
4. Only one Image can be uploaded per row. Any extras can be stored in the google drive folder. 5. The Weblink must be written in HTML format (text) in order for it to show up as a link in the database.
