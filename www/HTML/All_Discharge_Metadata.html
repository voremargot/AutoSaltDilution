Data Sources and Notes

DischargeID: This is an autogenerated value that is created when a new record is added.
EventID: This corresponds to the primary key in the autosalt summary table and identifies which event the discharge calculation belongs to.
SiteID: This corresponds to the primary key in the autosalt summary table and identifies which event the discharge calculation belongs to.
SensorID: This corresponds to the primary key in the sensors table and identifies which sensor the raw EC data is coming from.
CFID: This corresponds to the primary key in the calibration results table and identifies which CF value was used in the discharge calculations.
  • Please note that sensors shown in the calibration result table may not match with the SensorID in this table due to poor record keeping until May 2020.
Discharge: The calculated discharge value.
Uncertainty: The uncertainty of the discharge value.
Used:Indicates if this discharge value used in the calculation of the event's average discharge.

Units of Data

Discharge --> m3/s
Uncertainty --> %
Used --> Accepted Values: Y (yes), N (no)
