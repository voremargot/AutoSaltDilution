Data Sources and Notes

DischargeID (PK): This is an autogenerated value.
EventID (FK): Cooresponds to the primary key in the autosalt summary table and identifies which event the discharge calcualtion belongs to.
SiteID (FK): Cooresponds to the primary key in the autosalt summary table and identifies which event the discharge calcualtion belongs to. location
SensorID (FK): Cooresponds to the primary key in the sensors table and identifies which sensor the raw EC data is coming from for the calculated discharge.
CFID (FK): Cooresponds to the primary key in the calibration results table and identifies which CF value was used in the discharge calculations. Please note that sensors shown in the calibrtion result table may not match with the sensorID in this table due to confusion in the historical dataset up until May 2021.
Discharge: The calculated discharge value
Uncertianty: The uncertainty of the discahrge value
Used: Was this discharge value used in the calculation of average discharge for the event?

Units of Data

Discharge --> m3/s
Uncertainty --> %
Used --> Accepted Values: Y (yes), N (no)
