Data Sources and Notes

The data in this table is comes from the csv sheet that is filled out during the rating curve process

RCID: This is an autogenerated value that is created when a new record is added.
SiteID: This corresponds to the primary key in the site description table and tells us which site we are creating a rating curve for.
Version: The version of the rating curve that is being added.
Start Date: The oldest record of data included in the rating curve (YYYY-MM-DD).
End Date: The most recent record of data included in the rating curve (YYYY-MM-DD).
Shift: Indicates if a rating curve shift occur between the previous and current version.
Notes: General notes about the rating curve.
Link1 and Link2: A google drive link connecting to the summary of the rating curve.

Units of Data

Shift --> Accepted Values: Y (Yes), N (No)

Data Constraints

1. The start date must occur before the end date.
2. There cannot be multiple of the same version number per site (example: only one version 2 can exist at site 626).
3. To create a hyperlink for Link1 and Link2 , the google drive link must be written in html format.
