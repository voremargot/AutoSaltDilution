Data Sources and Notes

THE USER MUST UPDATE ALL COLUMNS IN THIS TABLE BY HAND

RCID: This is an autogenerated value that is created when a new record is added.
SiteID: The SiteID cooresponds to the SiteID column in the site_description table.
Version: This is the version of the rating curve that is being added.
Start Date: The oldest record of data included in the rating curve.
End Date: The most recent record of data included in the rating curve.
Shift: Records if a rating curve shift occur between the previous and current version.
Notes: General notes about the rating curve.
Link1 and Link2: A google drive link connecting to the summary of the rating curve.

Units of Data

Shift --> Accepted Values: Y (Yes), N (No)

Data Constraints

1. The start date must occur before the end date.
2. There cannot be mutiple of the same version number per site (example: only one version 2 can exist at site 626).
3. To create a hyperlink to Link1 and Link2 , the google drive link must be written in html format.
