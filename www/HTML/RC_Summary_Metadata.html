Data Sources and Notes

RCID (PK): This is an autogenerated value.
SiteID (FK): The Site ID cooresponds to one of the Site IDs in the site description table
Version: What version of the rating curve is being added
Start Date: The oldest record of data included in the rating curve
End Date: The newest record of data included in the rating curve
Shift: Did a rating curve shift occur between the previous and current version
Notes: General notes about the rating curve

Units of Data

Shift --> Accepted Values: Y (Yes), N (No)

Data Constraints

1. Start date must occur before the end date
2. There cannot be mutiple of the same version number per site (example: only one version 2 can exist at site 626)
