Data Sources and Notes

THE USER MUST UPDATE ALL COLUMNS IN THIS TABLE BY HAND

SiteID: This is the site name as an integer value.
Install Date: The date the site were installed (YYYY-MM-DD).
  • The install dates for sites 626, 703, 708, 819, 844, and 1015 are all estimated from data collection records
Lat: The latitude of the site location.
Lon: The longitude of the site location.
Elevation: The elevation of the site.
Width D: The width of the stream at the autosalt dumper site. This is obtained from a cross sectional survey of the streambed
Max Depth D: the maximum depth of the stream at the autosalt dumper site. This is obtained from a cross sectional survey of the streambed
Width EC: The width of the stream at the EC probe site. This is obtained from a cross sectional survey of the streambed
Max Depth EC: the maximum depth of the stream at the EC probe site. This is obtained from a cross sectional survey of the streambed
Slope: The stream gradient from the autosalt dumper to the EC probes.
Dist D EC: The downstream distance to the EC probes from the autosalt dumper.
Active: This specifies if the station is active.
Deactivation Date: This specifies when the station was deactivated.

Units of Data

Lat --> decimal degrees
Lon --> decimal degrees
Elevation --> m asl
Width D --> m
Max Depth D --> m
Width EC --> m
Max Depth EC --> m
Slope --> %
Dist D EC --> m
Active --> Accepted Values: Y (yes), N (no)

Data Constraints

1. All width and depth measurements must be positive values.
2. The install date must occur before the deactivation date.
3. If a site is marked as not active (i.e. Active =N) then a deactivation date must be listed.
