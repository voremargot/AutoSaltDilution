Data Sources and Notes

SiteId (PK): This is the name of the site as an integer value.
Install Date: Date the sites were installed. The install dates for sites 626, 703, 708, 819, 844, and 1015 are all estimated from data collection records
Lat: The latitude of the site location
Lon: The longitude of the site location
Elevation: Elevation of the autosalt dumper
Width D: Width of the stream at the autosalt dumper site. Obtained from a cross sectional survey of the streambed
Max Depth D: Maximum depth of the stream at the autosalt dumper site. Obtained from a cross sectional survey of the streambed
Width EC: Width of the stream at the autosalt EC probe site. Obtained from a cross sectional survey of the streambed
Max Depth EC: Maximum depth of the stream at the EC probe site. Obtained from a cross sectional survey of the streambed
Slope: Stream gradient from the autosalt dumper to the EC probes
Dist D EC: Downstream distance of the EC probes from the autosalt dumper
Active: Specifies if the station is active [Y= Yes, N=No]
Deactivation Date: Specifies when the station was deactivated.

Units of Data

Lat --> decimal degrees
Lon --> decimal degrees
Elevation --> m asl
Width D --> m
Max Depth D --> m
Width EC --> m
Max Depth EC --> m
Slope --> %
Dist D EC --> m

Data Constraints

1. All width and depth measurements must be positive
2. Install date must occur before the deactivation date
3. If a site is marked as not active (i.e. Active =N) then a deactivation date must be listed
