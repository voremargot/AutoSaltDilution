Data Sources and Notes

PeriodID (PK): This is an autogenerated value for each new row added
SiteID (FK): The Site ID cooresponds to one of the Site IDs in the site description table
Starting Date: The date when the barrel was refilled with new solution
Ending Date: Date when the next refill occurs

Data Constraints

1. Starting date must occur before the Ending date
2. The ending date of barrel period x will usually be the same as the starting date for period x+1
