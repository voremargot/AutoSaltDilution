Data Sources and Notes

SensorID (PK): This is an autogenerated value
SiteID (FK):The Site ID cooresponds to one of the Site IDs in the site description table
Probe Number: The probe number that will coorespond to the sensor in the raw data files
Sensor Type: The type of sensor used
Serial Number: The sensors serial number
River Loc: The location of the sensor in the river
Install Date: Date when the sensor was installed at the site
Deactivation Date: Date that the sensor was removed from the site

Units of Data

Sensor Type --> Accepted values: THRECS, WQ, Fathom
River Loc --> Accepted values: RR, RL, RR-L, RR-H, RL-L, RL-H, RC, Lab

RR-> river right, RL-> river left, RC-> river center, -L-> low, -H-> high


Data Constraints

1. Install date must occur before the deactivation date
2. Sensor type and river loc can only be recorded as one of the accepted values. If different values need to be added, the data type will need to be changed within the database scheme
