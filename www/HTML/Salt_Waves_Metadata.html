Data Sources and Notes

WaveID: This is an autogenerated value that is created when a new record is added.
Site ID: This comes from the primary key of the Autosalt Summary table and tells us which site the salt wave relates to.
EventID: This comes from the primary key of the Autosalt Summary table and tells us which event the salt wave relates to.
SensorID: This comes from the primary key of the sensors table and tells us which sensor the salt wave was seen in.
Start ECWave:The time when the EC wave started. This is determined by the code.
End ECWave: The time when the EC wave ended. This is determined by the code.
Time MaxEC: This is the time when the maximum EC value is reached.
StartingEC: This is the average EC value prior to the salt wave occurring.
EndingEC: This is the average EC after the salt wave has passed.
PeakEC: The maximum EC values that occurred during the salt wave.
Flags: Highlights common errors within the salt wave.
Comment: Specific notes regarding the salt wave.

Units of Data

StartingEC --> uS/cm
EndingEC --> uS/cm
PeakEC --> uS/cm

Flags

L -> The EC value is low (Average EC <5)
M -> Multiple peaks occur in the salt wave
N -> Minimal to no salt wave occurs (sensor may be out of the water)
S-> Salt wave has significant spikes
Sd -> Salt wave has high standard deviations making the data noisy
Pw-> There is a partial wave (missing parts of either the falling or rising limb)
T-> Timing of salt waves between two or more sensors is off (time of peak EC is different by more than 45sec).
