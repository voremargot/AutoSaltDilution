Data Sources and Notes

WaveID (PK): This is an autogenerated value.
Site ID (FK): This comes from the primary key of the Autosalt Summary table and tells us which site and event the salt wave relates to.
EventID (FK): This comes from the primary key of the Autosalt Summary table and tells us which event the salt wave relates to.
SensorID (FK): This comes from the primark key of the sensors table and tells us which sensor the salt wave occurred in
Start ECWave:The time when the EC wave started. this is determined in the code
End ECWave: The time when the EC wave ended. This is determined in the code
Time MaxEC: This is the time when the maximum EC value is reached.
StartingEC: This is the average EC value prior to the salt wave occurring.
EndingEC: This is the average EC after the salt wave has passed.
PeakEC: The maximum EC values that occur during the salt wave.
Flags: Highlights common errors and occurrances within the salt wave
Comment: Specific notes regarding the salt wave

Units of Data

StartingEC --> uS/cm
EndingEC --> uS/cm
PeakEC --> uS/cm

Flags

L -> The EC value is low (<10) M -> Multiple peaks occur in the salt wave
N -> Minimal to no salt wave occurs (sensor may be out of the water)
S-> Salt wave has significant spikes
SD -> Salt wave has high standard deviations making the data nosiy
P-> There is a partial wave (missing either falling or rising limb)
T-> Timing of salt waves at different sensors are off
Df -> Starting EC is more than 5units lower than other sensors
