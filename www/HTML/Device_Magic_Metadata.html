Data Sources and Notes

All data in this table is the raw data uploaded from the device magic form prior to it being organized into its repective tables.

DMID: This is an autogenerated value that is created when a new record is added.
Submitted: The date and time the form was submitted from the device (local time PST/PDT, YYYY-MM-DD).
Date Visit: The date the site was visited (YYYY-MM-DD).
Time Visit: The time the site was visited (local time PST/PDT).
SiteID:The site that was visited.
Technician and Technicians Other: List of technicians who visted the field.
Upstream Photo: Link to an upstream photo from the site.
Downstream Photo: Link to a downstream photo from the site.
Barrel Fill: Indicates of a reservoir refill was performed during the visit.
CF Event: Indicates if CF mesurements were taken during the visit.
EC Sensor Change: Indicates if there were any EC sensor changes made to the network during the visit.

Volume Solution:If a barrel fill occurred, this indicates how much solution was in the reservoir before it was refilled.
Salt Added : If a barrel fill occurred, this indicates how much salt was added to the new solution mixture.
Salt Added : If a barrel fill occurred, this indicates how much water was added to the new solution mixture.
Volume Depart : If a barrel fill occurred, this indicates how much solution mixture was in the reservior upon departure from the site.
Salt Remaining Site : If a barrel fill occurred, this indicates how much salt remains on the site before departure.
Barrel Fill Notes: If a barrel fill occurred, this contains any important note about the refill.

Time Barrel Period: If a CF measurement happened, this indicates when the measurements occurred in relation to the a barrel fill event.
Trials CF: If a CF measurement happened, this indicates how many trials were done for each measurement.

Action: If the sensor network was changed, this tells us what action was performed.
( Add a sensor to the network without replacement, Remove a sensor from the network without replacement, Replace a sensor in the network with a new sensor.

Reason: If the sensor network was changed, this indicates a reason for why the change occurred.

Sen R Removed Type (Other): If a sensor was replaced, this indicates what the removed sensor's type is.
Sen R Removed SN: If a sensor was replaced, this indicates what the removed sensor's serial number is.
Sen R New Type (Other): If a sensor was replaced, this indicates what the new sensor's type is.
Sen R New SN: If a sensor was replaced, this indicates the new sensor's serial number.
Sen R New RiverLoc (Other): If a sensor was replaced, this indicates the new sensor's location in the river.
Sen R New ProbeNum: If a sensor was replaced, this indicates the new sensor's probe number

Sen Remove Type (Other): If a sensor was removed, this indicates the sensor's type.
Sen Remove SN: If a sensor was removed, this indicates the sensor's serial number.
Sen Remove ProbeNum: If a sensor was removed, this indicates the sensor's probe number.

Sen Add Type: If a sensor was added, this indicates the sensor's type.
Sen Add RiverLoc (Other): If a sensor was added, this indicates where in the river the sensor was placed.
Sen Add ProbeNum: If a sensor was added, this indicates the sensor's probe number.

Notes Weather: General notes about the weather during the field visit.
Notes Repairs: General notes about repairs and adjustments that were made during the site visit.
Notes ToDo: General notes about things that need to be done at the site at future visits.
Notes Other: General notes about the visit that do not fit into the other note catagories.

Visit Added: Indicates if the visit information has been added to the Field Visit table.
Barrel Added: Indicates if the barrel fill information has been added to the Barrel Fill table.
Sensor Added: Indicates if the sensor network change information has been added to the Sensor table.
CF Added:Indicates if the CF values have been correctly entered into the Calibration Event and Results tables.

Units of Data

Volume Solution --> volume rounded down to nearest 25L
Salt Added --> kg
Water Added --> volume rounded down to nearest 25L
Volume Depart --> volume rounded down to nearest 25L
Salt Remaining Site --> kg
